gofmt 一统天下

gofmt 是目前golang里面用的最多的用来格式化代码风格的一个命令行工具， 很多知名项目都用它来保证自己的最基本的代码风格与官方和社区推荐的风格一致。

天下之外

其实标准gofmt有很多代码风格没有规定的地方比如以下几种场景：

  • 常量命名风格
  • 变量命名风格
  • 函数命名风格
  • 对象字段使用风格
  • 值传递，还是指针传递

据库对象：

  1. 可空字段，使用指针，用来表示NULL
  2. 对于不可空字段不使用指针类型， 用来表示这个字段一定有值
